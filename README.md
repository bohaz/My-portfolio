### Prerequisites

In order to run this project you need:

- Visual Studio Code installed
- Npm installed
- A Github account
- Git installed

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:bohaz/My-portfolio.git
```

To install all dependencies, run:

```npm install```
